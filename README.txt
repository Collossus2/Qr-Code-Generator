# Qr-Code-Generator
This application generates QR Codes using QR Python Package and Pillow. Feel free to try it out! The GUI is made with tkinter. Type any text to the text bar and hit Enter/Generate Button! Then view the project files and download your QR Code image via "more" (3 dots) button.

#Usage
1. Paste your YouTube link to the text Entry box
2. Hit ENTER or GENERATE QR
3. A QR Code will be placed in the project directory
